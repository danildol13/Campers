# Campers Next

A campervan rental catalog built with Next.js. Browse, filter, and book campervans from a curated collection across Ukraine.

## Features

- Browse campervans with pagination (4 per page)
- Filter by city, equipment (AC, kitchen, bathroom, TV, automatic), and vehicle type (Van, Fully Integrated, Alcove)
- Save favorites — persisted across page reloads
- Detailed camper page with Features and Reviews tabs
- Booking form with validation
- Loading states with Suspense and custom Loader

## Technologies

- [Next.js 15](https://nextjs.org) (App Router)
- [TypeScript](https://www.typescriptlang.org)
- [Zustand](https://zustand-demo.pmnd.rs) — state management with localStorage persistence
- [Axios](https://axios-http.com) — HTTP client
- [React Hot Toast](https://react-hot-toast.com) — notifications
- [React DatePicker](https://reactdatepicker.com) — date selection
- CSS Modules — scoped styling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
